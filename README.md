# Engagement Invitation RSVP Website

A lightweight event invitation website with RSVP tracking, payment instructions, and custom domain hosting.

This project was built to create a simple, elegant online invitation and reservation system for an engagement celebration while keeping costs minimal.

---

## Live Website

https://makayla-cory.party

---

## Project Overview

This website allows guests to:

* View the engagement invitation
* Confirm their attendance
* Submit RSVP information through a form
* Choose a payment method
* Allow the host to track reservations in a Google Sheet

The site is deployed using GitHub Pages and connected to a custom domain.

---

## Features

* Responsive invitation website
* RSVP form connected to Google Forms
* Guest tracking using Google Sheets
* Payment instructions for Cash App and Zelle
* Custom domain with HTTPS
* Hosted for free using GitHub Pages

## Deployment

The website is deployed using GitHub Pages.

Steps used:

1. Upload project files to GitHub repository
2. Enable GitHub Pages deployment from the main branch
3. Connect custom domain via DNS configuration
4. Enable HTTPS
